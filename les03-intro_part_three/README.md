### For loop

```javascript
    // do something 10 times
}
```

### While loop

```javascript
let i = 0;

while (i < 10) {
    // do something 10 times
    i++;
}
```

### Do while loop

```javascript
let j = 0;
do {
    // do something 10 times
    j++;
} while (j < 10);
```
